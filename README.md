# React-1

## 설치와 환경 설정
### 설치
- `npm install -g create-react-app`
- `yarn global add create-react-app`

### 새 프로젝트 생성
- `npx create-react-app my-app`

## 기본 문법과 주석 처리
### 기본 문법
- JSX 문법을 사용해서 UI를 작성함.
- JSX는 JavaScript의 확장 문법으로, JavaScript 코드 안에 HTML 코드를 작성할 수 있다.

### 주석 처리 방법
- JSX 내에서 주석은 중괄호 {/* */}와 같이 작성할 수 있다.
